# desktop

Tauri desktop application for the book-writing platform.

## Features

- Native desktop application
- Full access to file system
- Integration with Rust core library
- Cross-platform (Windows, macOS, Linux)

## Development

```bash
pnpm install
pnpm tauri dev
```

## Build

```bash
pnpm tauri build
```

## Tech Stack

- Tauri
- React 18
- TypeScript
- Vite
- React Router
- Rust backend with bookwriter-core
