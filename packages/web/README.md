# web

React web application for the book-writing platform.

## Features

- Book editor interface
- Chapter management
- Responsive design

## Development

```bash
pnpm install
pnpm dev
```

The app will be available at `http://localhost:3000`.

## Build

```bash
pnpm build
```

## Tech Stack

- React 18
- TypeScript
- Vite
- React Router
